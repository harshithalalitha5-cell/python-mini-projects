# Python Mini Projects (GUI + Algorithms)

This repository contains a menu-driven Python application that combines
multiple mini projects into a single file.

## Projects Included
1. Prime Number Checker & Visualizer (wxPython GUI)
2. ASCII Art Generator from Image (wxPython GUI)
3. Magic Square Generator (Odd Order)
4. Sudoku Generator (wxPython GUI)

## How to Run
```bash
python python_mini_projects_gui.py
